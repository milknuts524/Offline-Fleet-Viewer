｢提督手帳｣の使い方

提督手帳は、艦これの艦娘や装備、資材などをiPhoneで確認できるオフラインビューアです。
本アプリは非公式の艦隊データ閲覧アプリです。

艦これ公式サーバーへ接続したり、ゲームを操作したりする機能はありません。
ご自身の艦隊データをJSON形式で保存し、iPhoneで閲覧するためのアプリです。

今回は、提督手帳を使うための準備手順を紹介します。

準備するもの
iPhone（提督手帳をインストール）

艦これが動作するPC/Mac, Google Chrome

Python実行環境

データ変換スクリプト

作業時間は、コード作業など慣れた方なら5～10分程度です。
慣れてない方は、Chat GPTさんやClaudeさんに相談しながらやればなんとかなります。

Pythonについて
変換スクリプトの実行には Python を利用します。

といっても特別な開発環境は必要ありません。
Macでは標準で利用できる場合が多く、Windowsでも無料でインストールできます。

難しいプログラミング知識は不要で、ターミナル（またはコマンドプロンプト）から実行するだけです。

1. 艦これのデータを保存する
まずPC/Macで艦これを起動します。

ブラウザの開発者ツールなどを利用して、以下のデータを保存してください。

port.txt

start2.txt

slotitem.txt

useitem.txt

これらは艦これが内部で利用しているデータです。
わかんないよ、て方は、以前の記事もご参照下さい。

https://note.com/real_quoll1408/n/nb0b0aa67ec88


Chrome DevToolsでデータ取得
Chromeで艦これを開き、開発者ツールを使います。
手順は以下。

Chromeで艦これを起動

Option + Command + I でDevToolsを開く

Network タブを開く

Fetch/XHR に絞る

Preserve log をON

母港・編成・装備画面などを開く

必要なAPIレスポンスをコピーする

2. 変換スクリプトを実行する
変換スクリプトはこちら
https://github.com/milknuts524/Offline-Fleet-Viewer-Tools

使い方は、日本語/EnでReadmeにも書いてあります。

保存したデータと変換スクリプトを同じフォルダに配置します。
ターミナルでそのフォルダを開き、

python3 convert_kancolle.py

copy
を実行します。

正常に終了すると、以下のJSONファイルが生成されます。

ships.json

items.json

materials.json

useitems.json

payitems.json

3. JSONファイルをiPhoneへ転送する
生成された5つのJSONファイルを、

iCloud Drive

Dropbox

AirDrop

など、お好きな方法でiPhoneへ転送してください。
ファイルアプリから見える場所に保存しておくと便利です。

4. 提督手帳で読み込む
提督手帳を起動します。

艦娘
「読込」→「ships.json」

装備
「読込」→「items.json」

資材・道具
「読込」から

materials.json

useitems.json

payitems.json

をそれぞれ選択してください。

読み込みが完了すると、艦娘一覧、装備一覧、資材情報が表示されます。

よくある質問
Q. インターネット接続は必要ですか？
JSONを読み込んだ後は不要です。
提督手帳はオフラインで閲覧できます。

Q. 艦これアカウント情報は送信されますか？
送信されません。
提督手帳は外部サーバーへ艦隊データを送信しません。
データは端末内で閲覧されます。

Q. JSONファイルを間違えて選択した場合は？
対応するファイルを選択し直してください。

ships.json → 艦娘

items.json → 装備

materials.json → 資材

useitems.json → 特殊アイテム

payitems.json → 課金アイテム
